#### Java反射
##### 一：反射的定义
```
    Java中反射是指在运行过程中，对任意一个类，都能够知道这个类的所有方法和属性；对于任意一个对象，都能够调用它的任意一个方法和属性，这种动态获取信息以及动态调用对象的方法称为Java语言的反射机制。
 ```
##### 二：反射中获取Class对象的三种方式
```
	1.第一种(对象都有了还要反射干什么)
	Student stu=new Student();
	Class clazz=stu.getClass();
	 
	2：第二种(需要导入类的包 ，依赖太强，不导入就会抛出编译错误)
	Class clazz=Student.Class
	 
	3：第三种（推荐使用）
	Class clazz=Class.forName("类的全路径");
```	 
##### 三：通过Class对象调用构造方法并使用
```
    Constructor[] conArray = clazz.getConstructors(); //获取所有公有方法
    
    conArray = clazz.getDeclaredConstructors(); //获取所有方法（私有和公有）
    
    Constructor con = clazz.getConstructors(null);//获取无参构造方法
    con.newInstance();//执行无参构造方法

    Constructor con = clazz.getConstructors(int.class)//调用有参构造方法，参数为对应类型.class
    con.newInstance(1);//执行有参构造方法
 
    Constructor con = clazz.getDeclaredConstructors(int.class)//调用私有有参构造方法
    con.setAccessible(true);//暴力访问，忽略掉修饰符
    con.newInstance(1);//执行有参私有构造方法
```

 
##### 四：通过反射获取成员变量
```
    Field[] fields=clazz.getFields();//获取所有公有的字段
    Field[] fields=clazz.getDeclaredFields();//获取所有（公私）字段
    Field field=clazz.getField("name");//获取实体里面叫name的属性
    Object obj=clazz.getConstructor().newInstance();//获取一个对象
    field.set(obj,"Jack");//为name设置值叫Jack
    实体 别名=（实体）obj//把对象转化为实体  System.out.println(别名.name);

    Field f=clazz.getDeclaredField("age");//获取实体私有属性age
    Object obj=clazz.getConstructor().newInstance();//获取一个对象
    f.setAccessible(true);//暴力反射，解除私有限定
    f.set(obj,23);//为私有属性age设置值23
 ```
##### 五：获取成员方法
```
    Method[] m=clazz.getMethods();//获取所有公有方法

    Method[] m=clazz.getDeclaredMethods();//获取所有私有方法

    Method methods=clazz.getMethod("showName", String.class);//获取实体中叫showName的方法，参数为String类型
    
    Object obj=clazz.getConstructor().newInstance();//获取一个对象
    
    methods.invoke(obj,"测试");//为调用showName方法，传递参数“测试”

    Method m=clazz.getDeclaredMethod("show1");//获取无参私有方法show1
    m.setAccessible(true);//暴力访问，解除私有限定
    Object o=clazz.getConstructor().newInstance();//获取对象
    m.invoke(o);//执行
```

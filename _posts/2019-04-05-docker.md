##### 安装环境
```
 win10 专业版
 docker for windows 版本 19.03.13
 下载地址 https://www.docker.com/products/docker-desktop
```
##### 安装教程
```
 先下载docker for windows
```
![Docker](down.png)

```
点击安装，来到安装页面
如果你的win10 Hyper-V没有启动，安装的时候会帮你启动，安装完成需要重启电脑

WSL 2 是适用于 Linux 的 Windows 子系统可让开发人员直接在 Windows 上按原样运行 GNU/Linux 环境
（包括大多数命令行工具、实用工具和应用程序），且不会产生传统虚拟机或双启动设置开销。
```
![Docker](1.png)

```
重启电脑
```
![Docker](2.png)

```
重启Docker
```
![Docker](3.png)

```
启动失败，根据提示更新WSL2 Linux 内核
```
![Docker](4.png)

```
下载安装WSL2 Linux内核，下载完成后直接点击安装即可
```
![Docker](5.png)

```
安装成功后启动
```
![Docker](7.png)

```
修改配置文件位置，默认C盘
win10 搜索 Hyper-V ，打开应用，选择设置，更改虚拟硬盘和虚拟机位置
```
![Docker](pz.png)

 打开CMD或Windows Powershell 输入
 docker version 或者 docker --version 能看到版本号说明安装成功

##### 系统命令

```ruby
cat /etc/redhat-release 查询linux版本
cat /proc/cpuinfo| grep "cpu cores"| uniq  查看服务器CPU核数
cat /proc/cpuinfo| grep "physical id"| sort| uniq| wc -l CPU数
cat /proc/cpuinfo| grep "processor"| wc -l 逻辑CPU总数
uname -r  linux上查看系统版本
lsof -i:端口号 查看端口号使用情况
df -T 查看分区的文件系统
df -h 显示目前所有文件系统的可用空间及使用情形
du -sh *   查看当前目录下各个文件及目录占用空间大小
ls -lh  查看文件大小
free -h      命令是一个快速查看内存使用情况的方法
shutdown -r now 是关机后立即重新启动
reboot  是直接重新启动
date -R 查看时区
env 查看所有环境变量
vim /etc/hostname 或 hostnamectl set-hostname [NAME]  修改主机名
```

##### 修改文件权限和用户组


```ruby
第一个字符，如果是“-”则表示是文件，如果是“d”则表示是目录
r是4（读），w是2（写），x是1（执行），-是0
所以上面文件是6 4 4权限

修改权限命令
chmod 755 test.txt

chmod 755 文件或目录    不修改目录里面的文件
chmod  -R 755 目录    修改目录里面的文件
或者
chmod +wx 文件或目录
chmod -wx 文件或目录

chown 用户:用户组 文件名/文件夹  #修改指定文件用户
chown root:root test.txt
chown -R 用户:用户组 文件名/文件夹 #修改文件下所有的文件
```

##### Linux CPU问题排查

```ruby

  printf "%x\n" PID
```

##### Linux 常见参数修改

```ruby
vim /etc/ssh/sshd_config
MaxSessions：最大允许链接数，默认10.
MaxStartups：最大允许保持多少个未认证链接（未输入登录密码），默认值是10.

把他们的数字改大，例如改成
MaxSessions 1000
MaxStartups 1000

最后重启
sshd service sshd restart
systemctl restart sshd.service
```

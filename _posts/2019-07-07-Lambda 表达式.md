##### 用lambda表达式实现Runnable

```java
// Java 8之前：
new Thread(new Runnable() {
    @Override
    public void run() {
    System.out.println("Before Java8, too much code for too little to do");
    }

new Thread( () -> System.out.println("In Java8, Lambda expression rocks !!") ).start();
```


##### List集合中使用lambda表达式

```java
List<Integer> a=new ArrayList<>();
a.add(1);
a.add(2);
List<Integer> b=new ArrayList<>();
b.add(3);
b.add(4);
//合并两个List集合
List<Integer> figures= Stream.of(a,b).flatMap(u->u.stream()).collect(Collectors.toList());
figures.forEach(f->System.out.println(f));
//取出集合最大值
Integer m=figures.stream().max(Comparator.comparing(u->u)).get();

Integer s = figures.stream().reduce((x, y) -> x + y).get();
```

##### Java jdk1.8提供了的接口

```java
T参数，R返回值
Function<T,R>
class LambdaT{
    public static String print(String data){
       return data;
    }
    public static void main(String[] args) {
        Function<String,String> fun=LambdaT::print;
        System.out.println(fun.apply("A"));
    }
}

消费者，有参数，没返回值
Consumer<T>

class LambdaT{
    public static void print(String data){
        System.out.println(data);
    }
    public static void main(String[] args) {
        Consumer<String> fun=LambdaT::print;
        fun.accept("A");
    }
}

供给型接口，不需要参数，可以返回结果
Supplier<T>
class LambdaT{
    public static String print(){
        return "A";
    }
    public static void main(String[] args) {
        Supplier<String> fun=LambdaT::print;
        System.out.println(fun.get());
    }
}
```


##### Optional类
  
```java
一：作用
	解决空指针异常导致Java应用程序失败的问题
  二：方法
        Optional.of() 不允许参数是null，为空抛出异常
        Optional.ofNullable() 无限制
```

**1.方法：**

```java
isPresent()：判断值是否存在

void ifPresent(Consumer<? super T> consumer)
如果值存在则使用该值调用 consumer , 否则不做任何事情。

orElse(value)：如果optional对象保存的值不是null，则返回原来的值，否则返回value

orElseGet(Supplier supplier)：功能与orElse一样，只不过orElseGet参数是一个对象

filter(Predicate)：判断Optional对象中保存的值是否满足Predicate，并返回新的Optional。

map(Function)：对Optional中保存的值进行函数运算，并返回新的Optional(可以是任何类型)
```
**2.Optional用法**
```java
optional.ifPresent()可以判断返回值是否为空
ifPresent(Consumer<? super T> consumer)在结果部位空时进行消费操作
optional.get() 获取返回值

用Optional可以在没有值时指定一个返回值
Integer[] arr = new Integer[]{4,5,6,7,8,9};
Integer result = Stream.of(arr).filter(x->x>9).max(Comparator.naturalOrder()).orElse(-1);

Integer result1 = Stream.of(arr).filter(x->x>9).max(Comparator.naturalOrder()).orElseGet(()->-1);

Integer result2=Stream.of(arr).filter(x->x>9).max(Comparator.naturalOrder()).orElseThrow(RuntimeException::new); 

Optional.empty()创建一个空的Optional
Optional.of()创建指定值的Optional
map方法进行Optional的转换
Optional<Student> studentOptional = Optional.of(new Student("user1",21));
Optional<String> optionalStr = studentOptional.map(Student::getName);
flatMap方法进行Optional的迭代
Optional.of(参数).flatMap(Class::方法一).flatMap(Class::方法二);
```

##### lambda 收集器

```java
Collectors.maxBy()        找出最小值
Collectors.minBy()        找出最大值
Collectors.averagingInt() 求int类型平均值(averagingDouble,averagingLong)
Collectors.partitioningBy()数据分块
Collectors.groupingBy() 数据分组

eg:
public class test{
	private String name;
	private int age;
	//getter setter
}

Optional<Student> collect = list.stream().collect(maxBy(comparing(o -> o.getAge())));//找出最大年龄

Map<Boolean, List<Student>> collect = list.stream().collect(Collectors.partitioningBy(s -> s.getAge() < 30));//将数据按照大于30岁和小于30岁进行分组

字符串连接
String s = list.stream().map(x -> x.getName()).collect(Collectors.joining(",", "[", "]"));//把所有的名字用逗号连接起来[a,b,c]

根据名字分组，统计每组的的数量
Map<String, Long> map = list.stream().collect(groupingBy((x -> x.getName()),counting()));

根据年龄分组，获取每个组所有的人名字
Map<Integer, List<String>> map = list.stream().collect(groupingBy((x -> x.getAge()),mapping(Student::getName,toList())));

获取年龄大于10岁的人的名字，并且在控制台打印出来
list.stream().filter(o -> o.getAge() > 10).map(m -> m.getName()).peek(log->logger.info(log)).collect(Collectors.toSet());
```
